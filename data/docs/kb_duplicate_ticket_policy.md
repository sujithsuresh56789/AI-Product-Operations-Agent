# Duplicate Ticket Handling Policy

Before triaging a new ticket, check if an existing open ticket describes the same symptom on the same feature area (e.g., same page, same error).

If a duplicate is found:
1. Link the new ticket to the original as "duplicate of".
2. Post a short comment referencing the original ticket ID.
3. Close the duplicate; do not close the original.
4. Do not merge customer-identifying details between tickets.
